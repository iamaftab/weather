### How to use this weather app ?
Steps:
1. Goto openweathermap.org and Create an Account
2. From 'My API Keys' - copy your api key and paste it inside <kbd>weather.js</kbd> in this line 
   
   ```js 
   const APIKEY = 'Add your api';
   ```
